# Скрипты для выполнения рутинных задач

Простой скрипт для автоматической обработки Excel файлов.

Что делает:
- читает Excel файл
- показывает данные
- считает статистику (сумма, среднее, максимум, минимум)
- сохраняет результат в папку результаты

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python code_base/script.py
```

При первом запуске создастся пример файла и обработается.

## Использование своего файла

Открой script.py и измени путь к файлу в конце файла.
